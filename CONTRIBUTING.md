# contributing

## setup
- `pnpm install`
- `cp .env.example .env.local`
- `pnpm dev`

## expectations
- keep files under 500 lines
- prefer clear, descriptive names
- avoid defensive fallbacks unless they protect data or privacy
- run `pnpm lint` before opening a pr

## tests
- `pnpm test`
- `pnpm build`
