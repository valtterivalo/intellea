# examples

these examples show the supported integration surfaces. each folder contains minimal code you can drop into an existing project.

- `react/` render with GraphResponseRenderer
- `next-app-router/` use the renderer in a next.js app router page
- `web-component/` register and set `<intellea-graph>`
- `iframe/` postMessage into the embed shell
- `mcp/` call the mcp tool and render the response

shared payloads live in `examples/payloads/`.
